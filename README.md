# OpenCV
 <h4>Proje Klasörleri</h4><br>
 <p>
 Dersler klasöründe OpenCV'yi öğrenmek için gerekli araştırmalar yapılacak ve kodlar oluşturulacak.(videolar,makaleler,kitap...vs)</p>

<p>

</p>
<p>

</p>
